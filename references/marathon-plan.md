# 16-Week Marathon Training Plan

Structured plan based on zone training. Paces below are for sub-3 target — scale proportionally for other goals.

## Training Phases

| Phase | Weeks | Focus |
|-------|-------|-------|
| Base | 1-4 | Aerobic base + VO2max introduction, gradual mileage build |
| Build | 5-8 | Threshold work + race simulations, long run progression |
| Peak | 9-12 | Highest volume, longest runs, race-specific intensity |
| Taper | 13-16 | Volume reduction, maintain sharpness, race prep |

## Pace Zones (Sub-3 Reference)

Scale these proportionally for other target times.

| Zone | Description | Sub-3 Pace/km | HR % Max | Purpose |
|------|-------------|---------------|----------|---------|
| Z1 | Recovery/Easy | 5:15-6:00 | <60% | Recovery and aerobic base |
| Z2 | Aerobic Endurance | 4:45-5:15 | 60-70% | Endurance and fat metabolism |
| Z3 | Tempo/Marathon Pace | ~4:15 | 70-80% | Aerobic power and LT1 |
| Z4 | Threshold (LT2) | 4:00-4:10 | 80-90% | Lactate clearance and pacing |
| Z5 | VO2 Max | 3:45-3:55 | >90% | Maximum oxygen uptake |

## Weekly Structure (Typical)

| Day | Session Type |
|-----|-------------|
| Monday | Recovery run (Z1) |
| Tuesday | Key workout 1 (tempo/threshold/race sim) |
| Wednesday | Easy run (Z1-Z2) |
| Thursday | Key workout 2 (intervals/VO2max/hills) |
| Friday | Recovery + strides (Z1 + strides) |
| Saturday | Long run (Z2, with Z3 progression in later weeks) |
| Sunday | Rest or cross-train |

## Key Workout Types

### Tempo / Marathon Pace (Z3)
- Sustained efforts at marathon goal pace
- Develops muscular endurance and running economy
- Examples: 3x12min Z3, 3x20min Z3, 2x25min Z3

### Criss-Cross Tempo (Z3/Z4)
- Alternating between marathon pace and threshold
- Improves lactate management at race pace
- Examples: 6x6min Z3/Z4, 6x8min Z3/Z4

### Threshold (Z4)
- Hard, controlled efforts at lactate threshold
- Increases lactate clearance capacity
- Examples: 4x10min Z4, 5x8min Z4

### VO2 Max Intervals (Z5)
- Short, high-intensity intervals
- Raises aerobic ceiling so marathon pace feels easier
- Examples: 6x3min Z5, 8x2min Z5, hill repeats

### Race Simulation
- Extended efforts mimicking race conditions
- Practice pacing, fueling, mental strategies
- Examples: 1:45hrs (45min Z2 > 60min Z3), 2hrs (20min Z2 > 90min Z3 > 10min Z2)

### Strides
- 6-8x 20-30sec at fast controlled pace
- Improve running mechanics and neuromuscular activation
- Included weekly, typically on Friday

## Taper Guidelines

- Week 13: Reduce volume ~20%, maintain some intensity (Z3/Z4 work)
- Week 14: Reduce volume ~40%, shorter tempos
- Week 15: Reduce volume ~60%, minimal intensity
- Week 16: Race week, very light running + strides only

## Nutrition

### Daily
- Carbs: 50-60% of daily calories (whole grains, fruits, vegetables)
- Protein: 1.2-1.6 g/kg body weight (lean meats, fish, eggs, legumes)
- Fats: 20-30% of calories (avocados, nuts, seeds, olive oil)

### Pre-Run
- Under 60min: Light snack 30-60min before (banana, toast with peanut butter)
- Longer runs: Balanced meal 2-3hrs before (oatmeal with fruit, bagel with almond butter)

### During Long Runs (>90min)
- 30-60g carbs per hour (gels, chews, sports drinks)
- 400-800ml fluids per hour

### Post-Run
- Refuel within 30min: carbs + protein 3:1 ratio
- Rehydrate with water or electrolyte drinks

### Carb Loading (Race Week)
- 2-3 days before race: 8-10g carbs per kg body weight per day
- Low-fiber carb sources: white rice, pasta, potatoes, bananas

### Race Morning
- High-carb breakfast 2-3hrs before (toast with jam, oatmeal, bagel)
- Stick to tested foods only

## Recovery

- Sleep: 7-9 hours per night, naps after hard sessions
- Active recovery: Z1 runs, light cross-training (cycling, swimming, yoga)
- Hydration: Monitor urine color, add electrolytes after long runs
- Foam rolling: 3-5x per week
- Listen to body: Rest or modify if unusually tired

## Strength Training

- Frequency: 2x/week in base/build, 1x/week in peak, stop in taper
- Key exercises: Squats, lunges, deadlifts, step-ups, planks, calf raises
- Timing: After easy runs or rest days, never before key workouts
- Allow 48hrs between strength sessions
