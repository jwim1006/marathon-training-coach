# 16-Week Marathon Training Plan — Beginner (4:00-4:30+ Target)

For first-time marathoners or runners targeting 4:00-4:30+. Conservative approach prioritizing finishing healthy over time goals.

Target pace: ~5:41/km (4:00:00) to ~6:24/km (4:30:00). Adjust to athlete's comfort level.

## Pace Zones (4:30 Reference)

| Zone | Description | Pace/km | HR % Max | Purpose |
|------|-------------|---------|----------|---------|
| Z1 | Recovery/Walk | 7:30-8:30 | <60% | Recovery, walk breaks OK |
| Z2 | Easy Run | 6:45-7:30 | 60-70% | Aerobic base (most runs here) |
| Z3 | Tempo/Marathon Pace | ~6:24 | 70-80% | Marathon pace practice |
| Z4 | Threshold | 6:00-6:15 | 80-90% | Lactate threshold (used sparingly) |
| Z5 | VO2 Max | 5:40-5:55 | >90% | Short intervals only (optional) |

## Weekly Structure (Typical)

| Day | Session Type |
|-----|-------------|
| Monday | Rest |
| Tuesday | Easy run (Z2) |
| Wednesday | Easy run or cross-train |
| Thursday | Key workout (tempo or intervals, starting week 5) |
| Friday | Rest |
| Saturday | Long run (Z2) |
| Sunday | Easy run or rest |

4-5 runs per week. 30-55km peak weekly volume. Walk breaks are OK and encouraged.

**Walk breaks are fine** during long runs, especially in early weeks. A common strategy: run 10min, walk 1min. Reduce walk breaks as fitness builds.

## Key Workout Types

### Easy Runs (Z2)
- The foundation. Should feel comfortable and conversational.
- If you can't hold a conversation, slow down.
- Walk breaks are perfectly OK, especially early in the plan.

### Tempo / Marathon Pace (Z3)
- Introduced in week 5, kept short and manageable
- Builds comfort at race pace
- Examples: 2x8min, 2x10min, 3x8min Z3

### Strides
- 4-6x 20sec at fast controlled pace, full recovery between
- Improve running mechanics without adding fatigue
- Introduced from week 2

### Race Simulation (Peak Phase Only)
- Practice race-day fueling and pacing during long runs
- Last 10-15min of long run at marathon pace

## Important Notes for Beginners

1. **Walk breaks are not failure.** Many successful marathoners use run/walk strategies.
2. **Slow down your easy runs.** The #1 beginner mistake is running too fast too often.
3. **Don't skip recovery weeks.** They feel easy — that's the point.
4. **Practice race nutrition.** Start taking gels during long runs from week 9.
5. **Listen to your body.** Sharp pain = stop. General fatigue = OK to run easy.
6. **Cross-training counts.** Cycling, swimming, or walking on easy days builds fitness safely.
7. **Your only goal for race #1 is to finish.** Time goals can come later.

## Taper Guidelines

- Week 13: Reduce volume ~20%, keep 1 short tempo
- Week 14: Reduce volume ~40%, minimal intensity
- Week 15: Reduce volume ~60%, just strides
- Week 16: Race week, very light running only
