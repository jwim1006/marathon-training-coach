# 16-Week Marathon Training Plan — Sub-3:30 (3:15-3:30 Target)

Target pace: ~4:58/km (3:30:00) to ~4:37/km (3:15:00). Scale proportionally within this range.

## Pace Zones (3:30 Reference)

| Zone | Description | Pace/km | HR % Max | Purpose |
|------|-------------|---------|----------|---------|
| Z1 | Recovery/Easy | 6:00-6:45 | <60% | Recovery and aerobic base |
| Z2 | Aerobic Endurance | 5:30-6:00 | 60-70% | Endurance and fat metabolism |
| Z3 | Tempo/Marathon Pace | ~4:58 | 70-80% | Aerobic power and LT1 |
| Z4 | Threshold (LT2) | 4:40-4:50 | 80-90% | Lactate clearance and pacing |
| Z5 | VO2 Max | 4:20-4:35 | >90% | Maximum oxygen uptake |

## Weekly Structure (Typical)

| Day | Session Type |
|-----|-------------|
| Monday | Rest or recovery (Z1) |
| Tuesday | Key workout 1 (tempo/threshold) |
| Wednesday | Easy run (Z1-Z2) |
| Thursday | Key workout 2 (intervals/hills) |
| Friday | Recovery + strides (Z1 + strides) |
| Saturday | Long run (Z2, with Z3 progression in later weeks) |
| Sunday | Rest or cross-train |

5-6 runs per week. 50-75km peak weekly volume.

## Key Workout Types

### Tempo / Marathon Pace (Z3)
- Sustained efforts at marathon goal pace (~4:58/km)
- Examples: 3x10min, 3x15min, 2x20min Z3

### Threshold (Z4)
- Hard, controlled efforts at lactate threshold (~4:40-4:50/km)
- Examples: 3x10min, 4x8min Z4

### VO2 Max Intervals (Z5)
- Short, high-intensity intervals (~4:20-4:35/km)
- Examples: 6x2min, 5x3min Z5, hill repeats

### Race Simulation
- Extended efforts mimicking race conditions
- Examples: 1:30 (40min Z2 > 50min Z3), 1:45 (30min Z2 > 60min Z3 > 15min Z2)

## Taper Guidelines

- Week 13: Reduce volume ~20%, maintain some Z3/Z4
- Week 14: Reduce volume ~40%, shorter tempos
- Week 15: Reduce volume ~60%, minimal intensity
- Week 16: Race week, very light running + strides only
