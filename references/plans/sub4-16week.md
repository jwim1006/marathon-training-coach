# 16-Week Marathon Training Plan — Sub-4 (3:45-4:00 Target)

Target pace: ~5:41/km (4:00:00) to ~5:19/km (3:45:00). Scale proportionally within this range.

## Pace Zones (4:00 Reference)

| Zone | Description | Pace/km | HR % Max | Purpose |
|------|-------------|---------|----------|---------|
| Z1 | Recovery/Easy | 6:45-7:30 | <60% | Recovery and aerobic base |
| Z2 | Aerobic Endurance | 6:15-6:45 | 60-70% | Endurance and fat metabolism |
| Z3 | Tempo/Marathon Pace | ~5:41 | 70-80% | Aerobic power and LT1 |
| Z4 | Threshold (LT2) | 5:20-5:30 | 80-90% | Lactate clearance and pacing |
| Z5 | VO2 Max | 5:00-5:15 | >90% | Maximum oxygen uptake |

## Weekly Structure (Typical)

| Day | Session Type |
|-----|-------------|
| Monday | Rest |
| Tuesday | Key workout (tempo/threshold) |
| Wednesday | Easy run (Z1-Z2) |
| Thursday | Easy run or intervals |
| Friday | Rest or recovery |
| Saturday | Long run (Z2) |
| Sunday | Easy run or cross-train |

5 runs per week. 40-65km peak weekly volume.

## Key Workout Types

### Tempo / Marathon Pace (Z3)
- Sustained efforts at marathon goal pace (~5:41/km)
- Examples: 3x8min, 3x12min, 2x15min Z3

### Threshold (Z4)
- Hard, controlled efforts at lactate threshold (~5:20-5:30/km)
- Examples: 2x10min, 3x8min Z4

### VO2 Max Intervals (Z5)
- Short, high-intensity intervals (~5:00-5:15/km)
- Examples: 5x2min, 4x3min Z5, hill repeats

### Race Simulation
- Extended efforts mimicking race conditions
- Examples: 1:30 (30min Z2 > 45min Z3 > 15min Z2)

## Taper Guidelines

- Week 13: Reduce volume ~20%, maintain some Z3/Z4
- Week 14: Reduce volume ~40%, shorter tempos
- Week 15: Reduce volume ~60%, minimal intensity
- Week 16: Race week, very light running + strides only
