Pick the plan closest to the athlete's target time, then personalize using their zones, schedule, and current fitness from athlete_context.md.

## How to Personalize

1. Read `athlete_context.md` for foundation, form, schedule, goals
2. Pick closest plan from table above
3. Read that plan file
4. Adjust: swap days to match schedule, scale zones to athlete's HR/pace, modify volume based on current fitness and foundation
5. Discuss with athlete before finalizing

## Scaling Guide

Each plan provides pace zones for its target time. To scale:
- Calculate target marathon pace from finish time: `target_time / 42.195 = pace/km`
- Z3 (marathon pace) = target pace
- Z2 (easy) = target pace + 45-75s/km
- Z4 (threshold) = target pace - 15-25s/km
- Z5 (VO2max) = target pace - 30-45s/km
- Z1 (recovery) = target pace + 75-90s/km
